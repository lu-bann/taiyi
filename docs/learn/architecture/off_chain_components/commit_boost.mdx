### Taiyi-Boost

Taiyi-Boost is a [`custom PBS module`](https://commit-boost.github.io/commit-boost-client/developing/custom-modules) that extends the default PBS module of commit boost. It enables local block building if a builder fails to adhere to the constraints for a slot.
